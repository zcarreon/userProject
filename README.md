# Create a User

`````
- Open a wamp/mamp server
- Import db_movies.sql into your database
- go to this url : localhost/admin/admin_usercreate.php
- You can create a user, and their information will be stored in the database along with an encrypted, auto-generated password
`````

* Known Bugs 
`````
- Users passwords cannot be un-encrypted. to login with a new user, go into the database and copy the encypted password and use that. It should work.
`````
